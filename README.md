
# Network Administration - Week 3

## Project Overview
This project demonstrates a hybrid routing environment implemented in GNS3 using:

- OSPF
- RIP Version 1 and Version 2
- Route Redistribution
- Static Routing
- Floating Static Routes
- Failover Testing

## Software Used
- GNS3
- Cisco IOS
- VPCS

## Features
- OSPF neighbor establishment
- RIP v1 and RIP v2 comparison
- OSPF-RIP redistribution
- Static and floating static routing
- Network failover verification
- End-to-end connectivity testing

## Repository Contents
- GNS3 topology
- Router configurations
- Assignment report
- Verification screenshots
